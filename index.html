About & OS Journey

My OS Journey

  • 2001 — First computer: Packard Bell D160 with built‑in speaker monitor, running Windows 95. First app I played with: QuickTime 5.0 (from a Kid’s Toy CD).
  • 2004 — Moved to Microsoft Windows XP and later skinned it to look like Windows Vista.
  • 2006 — Built my first MS‑DOS application: a Pong‑style game.
  • 2007 — Discovered OSS and dove into Linux, starting with Ubuntu 7.04 (GNOME). Mirrors: Ubuntu • GNOME.
  • 2007 — Mac world: loved Unix‑like systems and the iPod; used FlyAKiteOSX, explored Mac OS X 10.5 Leopard Dashboard widgets. iPod nostalgia: ipod-classic-js.
  • Today — Dabbling with code again after years of web and GUI design.
